# geoprocessamento-do-matatu
Este repositório foi criado com a finalidade de fornecer mapas e fomentar o conhecimento, desenvolvidos para uma pesquisa Socioeconômica e Geoespacial do Bairro do Matatu, em Salvador - BA. Vinculado ao Projeto Cidades Internas da Universidade Salvador (UNIFACS) e com o apoio da Coordenação de Aperfeiçoamento de Pessoal de Nível Superior (CAPES).
